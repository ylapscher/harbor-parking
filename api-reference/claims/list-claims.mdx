---
title: 'List Claims'
api: 'GET /claims'
description: 'Retrieve parking spot claims with optional filtering'
---

Retrieve a list of parking spot claims. You can filter the results by claimer ID, availability ID, or claim status.

## Request

<RequestExample>
```bash
curl -X GET "https://parking.lapscher.com/api/claims" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
</RequestExample>

### Query Parameters

<ParamField query="claimer_id" type="string" format="uuid" required={false}>
  Filter claims by the ID of the user who made the claim
</ParamField>

<ParamField query="availability_id" type="string" format="uuid" required={false}>
  Filter claims by the availability ID
</ParamField>

<ParamField query="status" type="string" required={false}>
  Filter claims by status. Valid values: `pending`, `confirmed`, `released`, `expired`, `cancelled`
</ParamField>

### Response Fields

<ResponseField name="claims" type="array">
  Array of claim objects
</ResponseField>

<ResponseField name="claims[].id" type="string" format="uuid">
  Unique identifier for the claim
</ResponseField>

<ResponseField name="claims[].availability_id" type="string" format="uuid">
  ID of the availability being claimed
</ResponseField>

<ResponseField name="claims[].claimer_id" type="string" format="uuid">
  ID of the user who made the claim
</ResponseField>

<ResponseField name="claims[].status" type="string">
  Current status of the claim. Possible values: `pending`, `confirmed`, `released`, `expired`, `cancelled`
</ResponseField>

<ResponseField name="claims[].notes" type="string">
  Optional notes provided by the claimer
</ResponseField>

<ResponseField name="claims[].created_at" type="string" format="date-time">
  Timestamp when the claim was created
</ResponseField>

<ResponseField name="claims[].updated_at" type="string" format="date-time">
  Timestamp when the claim was last updated
</ResponseField>

<ResponseField name="claims[].availabilities" type="object">
  Associated availability information
</ResponseField>

<ResponseField name="claims[].availabilities.parking_spots" type="object">
  Associated parking spot information
</ResponseField>

<ResponseField name="claims[].availabilities.parking_spots.profiles" type="object">
  Owner profile information
</ResponseField>
