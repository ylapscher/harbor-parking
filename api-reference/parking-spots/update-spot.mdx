# Update Parking Spot

Update details of a parking spot owned by the authenticated user.

## Endpoint

```
PUT /parking-spots
```

## Authentication

This endpoint requires authentication. Include your JWT token in the Authorization header:

```
Authorization: Bearer YOUR_JWT_TOKEN
```

## Request Body

```json
{
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "spot_number": "A-15",
  "location": "Level 2 Parking Garage",
  "notes": "Near elevator"
}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string | Yes | ID of the parking spot to update |
| `spot_number` | string | No | New spot number or identifier |
| `location` | string | No | New location description |
| `notes` | string | No | New notes about the spot |

## Response

### Success Response (200 OK)

```json
{
  "parking_spot": {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "spot_number": "A-15",
    "location": "Level 2 Parking Garage",
    "notes": "Near elevator",
    "owner_id": "456e7890-e89b-12d3-a456-426614174001",
    "is_verified": true,
    "is_active": true,
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T11:45:00Z"
  }
}
```

### Error Responses

#### 400 Bad Request

```json
{
  "error": "Validation failed",
  "details": {
    "spot_number": {
      "_errors": ["Spot number must be at least 2 characters"]
    }
  }
}
```

#### 401 Unauthorized

```json
{
  "error": "Authentication required"
}
```

#### 403 Forbidden

```json
{
  "error": "You can only update your own parking spots"
}
```

#### 404 Not Found

```json
{
  "error": "Parking spot not found"
}
```

#### 409 Conflict

```json
{
  "error": "Spot number already exists"
}
```

## Example

### cURL

```bash
curl -X PUT "https://parking.lapscher.com/api/parking-spots" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "location": "Level 1 Parking Garage",
    "notes": "Near main entrance"
  }'
```

### JavaScript

```javascript
const response = await fetch('https://parking.lapscher.com/api/parking-spots', {
  method: 'PUT',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    id: '123e4567-e89b-12d3-a456-426614174000',
    location: 'Level 1 Parking Garage',
    notes: 'Near main entrance'
  })
});

const data = await response.json();
```

### Python

```python
import requests

response = requests.put(
    'https://parking.lapscher.com/api/parking-spots',
    headers={
        'Authorization': f'Bearer {token}',
        'Content-Type': 'application/json'
    },
    json={
        'id': '123e4567-e89b-12d3-a456-426614174000',
        'location': 'Level 1 Parking Garage',
        'notes': 'Near main entrance'
    }
)

data = response.json()
```

## Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique spot identifier |
| `spot_number` | string | Spot number or identifier |
| `location` | string | Location description |
| `notes` | string | Optional notes about the spot |
| `owner_id` | string | ID of the spot owner |
| `is_verified` | boolean | Whether the spot is verified by admin |
| `is_active` | boolean | Whether the spot is active |
| `created_at` | string | Creation timestamp |
| `updated_at` | string | Last update timestamp |

## Notes

- Only the spot owner can update their parking spots
- At least one field must be provided in the request body
- The `updated_at` timestamp is automatically updated
- Spot numbers must be unique within the system
- All timestamps are in ISO 8601 format
