# List Parking Spots

Retrieve a list of parking spots owned by the authenticated user.

## Endpoint

```
GET /parking-spots
```

## Authentication

This endpoint requires authentication. Include your JWT token in the Authorization header:

```
Authorization: Bearer YOUR_JWT_TOKEN
```

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `limit` | number | No | Number of spots to return (default: 50, max: 100) |
| `offset` | number | No | Number of spots to skip for pagination (default: 0) |
| `verified` | boolean | No | Filter by verification status |
| `active` | boolean | No | Filter by active status |

## Response

### Success Response (200 OK)

```json
{
  "parking_spots": [
    {
      "id": "123e4567-e89b-12d3-a456-426614174000",
      "spot_number": "A-15",
      "location": "Level 2 Parking Garage",
      "notes": "Near elevator",
      "owner_id": "456e7890-e89b-12d3-a456-426614174001",
      "is_verified": true,
      "is_active": true,
      "created_at": "2024-01-15T10:30:00Z",
      "updated_at": "2024-01-15T10:30:00Z"
    }
  ],
  "total": 1,
  "limit": 50,
  "offset": 0
}
```

### Error Responses

#### 401 Unauthorized

```json
{
  "error": "Authentication required"
}
```

#### 403 Forbidden

```json
{
  "error": "Account not approved"
}
```

## Example

### cURL

```bash
curl -X GET "https://parking.lapscher.com/api/parking-spots?limit=10&verified=true" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### JavaScript

```javascript
const response = await fetch('https://parking.lapscher.com/api/parking-spots?limit=10', {
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

const data = await response.json();
```

### Python

```python
import requests

response = requests.get(
    'https://parking.lapscher.com/api/parking-spots',
    headers={
        'Authorization': f'Bearer {token}'
    },
    params={
        'limit': 10,
        'verified': True
    }
)

data = response.json()
```

## Response Fields

### Parking Spots Array

Each spot object contains:

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique spot identifier |
| `spot_number` | string | Spot number or identifier |
| `location` | string | Location description |
| `notes` | string | Optional notes about the spot |
| `owner_id` | string | ID of the spot owner |
| `is_verified` | boolean | Whether the spot is verified by admin |
| `is_active` | boolean | Whether the spot is active |
| `created_at` | string | Creation timestamp |
| `updated_at` | string | Last update timestamp |

### Pagination Fields

| Field | Type | Description |
|-------|------|-------------|
| `total` | number | Total number of spots |
| `limit` | number | Number of spots returned |
| `offset` | number | Number of spots skipped |

## Notes

- Only returns parking spots owned by the authenticated user
- Results are sorted by creation date (newest first)
- The `verified` parameter filters spots by admin verification status
- The `active` parameter filters spots by active status
- All timestamps are in ISO 8601 format
