# Delete Availability

Remove an availability window owned by the authenticated user.

## Endpoint

```
DELETE /availabilities
```

## Authentication

This endpoint requires authentication. Include your JWT token in the Authorization header:

```
Authorization: Bearer YOUR_JWT_TOKEN
```

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string | Yes | ID of the availability to delete |

## Response

### Success Response (200 OK)

```json
{
  "message": "Availability deleted successfully"
}
```

### Error Responses

#### 401 Unauthorized

```json
{
  "error": "Authentication required"
}
```

#### 403 Forbidden

```json
{
  "error": "You can only delete availabilities for your own spots"
}
```

#### 404 Not Found

```json
{
  "error": "Availability not found"
}
```

#### 409 Conflict

```json
{
  "error": "Cannot delete availability with active claims"
}
```

## Example

### cURL

```bash
curl -X DELETE "https://parking.lapscher.com/api/availabilities?id=789e0123-e89b-12d3-a456-426614174002" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

### JavaScript

```javascript
const response = await fetch('https://parking.lapscher.com/api/availabilities?id=789e0123-e89b-12d3-a456-426614174002', {
  method: 'DELETE',
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

const data = await response.json();
```

### Python

```python
import requests

response = requests.delete(
    'https://parking.lapscher.com/api/availabilities',
    headers={
        'Authorization': f'Bearer {token}'
    },
    params={
        'id': '789e0123-e89b-12d3-a456-426614174002'
    }
)

data = response.json()
```

## Notes

- Only the spot owner can delete availabilities for their spots
- Availabilities with active claims cannot be deleted
- All associated claims will be automatically cancelled
- This action is irreversible
